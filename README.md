# c_lang_study

 git 브랜치 사용법

[설치}
1. git bash프로그램 설치 : 구글에서 git 검색후 설치

[세팅]
2. git bash 프롬프트창 열어서 아래 명령어 입력
git config --global user.email "git 닉네임"
git config --global user.name "git 이메일주소"

[branch이동]
git clone 깃주소

git checkout 브런치명

[branch업로드]
git add .(점)

git commit -m "업데이트메세지"

git push origin 브랜치명
